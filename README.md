## 📌 Project Overview
This project analyzes sales analysis using a real-world dataset. The objective is to evaluate

Total Orders

Total Quantity

Total Amount

Avg. Rating

Avg. Days to Deliver




## 📊 Dataset Summary
**Data Type:** sales data.

**Columns Used:**
TX ID,

Product,

Quantity,

Unit Price,

Amount,

Order Date,

Ship Date,

Customer Gender,

Order Mode,

Rating C,

State,

County,

Days to deliver,

Weeknum,

Gender Value.
